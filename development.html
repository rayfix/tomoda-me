Development Notes

These web pages use Middleman a static page generator with Zurb Foundation.

The source (and website) are hosted on GitHub at https://github.com/rayfix/tomoda-me

Making Changes

To make changes to the site:

  • Signup with GitHub
  • Fork the repository
  • Make changes
  • Test on local machine
  • Submit pull request

Setting up your Local Machine

Clone the repository and follow the instructions in the README file. As of this writing, I am using: 
$ bundle exec middleman version
Middleman 3.1.6

$ ruby --version
